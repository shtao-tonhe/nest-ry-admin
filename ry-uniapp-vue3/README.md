## uniapp 小程序服务
技术栈：uniapp + uni-ui + sass + vue3 + pinia + typescript

## 项目描述
本项目参考: RuoYi-App项目改造
新增无感刷新token功能
仅搭建一个架子，后续新功能可自定义开发

## 步骤

1.导入模块
```bash
# install
$ npm install
```

2.启动：
配置 manifest.json 微信小程序配置
使用HbuildX打开 - 运行 - 运行到小程序 - 微信开发者工具(配置相关端口以及安全设置)打开

3.发布
配置 manifest.json 微信小程序配置
使用HbuildX打开 - 发布 - 小程序-微信 - 微信开发者工具发布
